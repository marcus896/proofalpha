# Maintainers

ProofAlpha is currently maintained by the repository owner.

Public maintainer identity and contact details should be taken from the repository profile and security settings rather than duplicated in this file.

## Responsibilities

Maintainers:

- review pull requests and triage issues;
- manage releases and dependency updates;
- enforce the open-source and security boundary;
- respond to private vulnerability reports;
- preserve paper-first and evidence-first defaults;
- reject unverifiable performance claims and unsafe execution changes.

## Adding maintainers

New maintainers are added through sustained, reviewable contributions and an explicit governance decision. Access must follow least-privilege principles.
