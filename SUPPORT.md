# Support

Use GitHub Issues for reproducible bugs and GitHub Discussions for design questions once those channels are enabled.

Support covers:

- installation and packaging;
- documented workflows;
- reproducible defects;
- plugin and schema questions;
- security-safe configuration.

Support does not include financial advice, strategy recommendations, profit expectations, account recovery, exchange disputes, or private production deployment.

For security vulnerabilities, follow `SECURITY.md` and do not post publicly.
