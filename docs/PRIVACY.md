# Privacy

ProofAlpha is local-first and has no default remote telemetry.

The project must not collect or transmit:

- exchange credentials;
- account balances, positions, or order history;
- strategy text or proprietary parameters;
- user datasets;
- persistent device identifiers;
- private local paths;
- model prompts containing secrets.

Logs, evidence cards, and bundles require redaction tests. Any future optional analytics must be opt-in, aggregated, documented, and independently disableable.
