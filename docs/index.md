# ProofAlpha Documentation

ProofAlpha is an AI agent that builds, backtests, and keeps improving crypto trading strategies. Give it an idea and it generates many variations, backtests each against realistic market conditions, learns from the results, and iterates until it has a strong, well-validated candidate — with a full, reproducible report.

## What makes it different

- An autonomous **idea → backtest → improve → compare** loop, not just a one-shot backtester.
- Realistic backtesting with fees, funding, slippage, latency, and partial fills.
- Walk-forward, stress, and overfit validation so only strong candidates advance.
- Reproducible evidence artifacts: run cards, dashboards, and evidence cards.
- Paper/no-key by default, with research and live execution kept cleanly separate.
