# ProofAlpha Brand

## Identity

- Product name: ProofAlpha
- Descriptor: Autonomous Crypto Strategy Engine
- Tagline: Evidence before execution.
- Repository slug: `proofalpha`
- Package name: `proofalpha`
- Console command: `proofalpha`

ProofAlpha is a public, evidence-first research and paper-trading project. Public surfaces should emphasize validation, reproducibility, paper/no-key defaults, and risk-gated execution readiness. Do not imply guaranteed profit, passive income, live trading approval, or investment advice.

## Voice

Use direct, technical, safety-first language.

Preferred phrases:

- Evidence before execution.
- Paper mode by default.
- No private keys required.
- Validation-first research.
- Risk-gated execution readiness.

Avoid:

- Guaranteed returns
- Passive income
- Fully automated profit
- Live trading by default
- Advice to trade real capital
